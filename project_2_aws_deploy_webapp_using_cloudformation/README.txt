## Prerequisites

1.) Provide a Key Value Pair with the name "ssh_ec2_instance" which is used by the bastion host Ec2 instances.
2.) Run the aws client with user credentials, that allow to automatically create IAM role and instance profiles.
3.) Configure the parameter files to your needs (MyIP ect.)
